Help support Autism awareness by putting a Fathers with an Autistic Child Ribbon on your WordPress site or network.


== Description ==

A small ribbon with a puzzle logo and the words "Proud Father with an Autistic Child" linking to http://www.facebook.com/Fatherswithchildrenwithautism) Facebook page.
== Installation ==


1. Download the plugin zip archive.

2. Extract it in your `wp-content/plugins` folder or upload via admin panel.

3. Browse to your plugins section and active the plugin.
